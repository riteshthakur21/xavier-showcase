Capabilities

Strict Verification

Smart Email Alerts

Integrated with Brevo API to send real-time transactional emails for account approvals and password resets.

The Stack

Modern Tech Architecture

We chose a battle‑tested, modern stack to ensure the platform is lightning fast, fully type‑safe, and highly scalable. Deployed on Vercel and Render for optimal edge performance.

Next.js 14 & Tailwind

App Router, Server Components, and utility‑first styling.

Node.js & Express API

Robust REST API with custom JWT Auth middleware.

Prisma ORM & Neon Postgres

Strictly typed database schema hosted on serverless Postgres.

Developer Setup

Clone the repo and spin it up in minutes — copy the commands below.

bash — ~/xavier
➜ ~ git clone https://github.com/riteshthakur21/xavier-alumni-connect.git
➜ ~ cd xavier-alumni-connect
# Backend server
➜ backend npm install && npx prisma db push
➜ backend npm run dev
# Frontend (new terminal)
➜ frontend npm install && npm run dev
Server running on http://localhost:3000 ✨
